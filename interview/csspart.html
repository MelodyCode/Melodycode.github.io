# margin（外边距）纵向重叠的问题

 - 相邻元素垂直方向(margin-top 与 margin-left)外边距会发生重叠现象
 - 空白内容的`<p></p>`也会重叠
 - 兄弟元素：
	+ 兄弟元素间的相邻垂直外间距会取两者之间值较大的（两者都是正值）
	+ 特殊情况：如果相邻的外间距一正一负，取两者的和；若均为负值，则取两者中绝对值较大的
	+ 兄弟元素外间距的重叠对开发有利，不需要处理
  - 父子元素：
	+ 父子元素间的*相邻外间距*，子元素会传递给父元素

<!-- AAA与BBB间距：15px-->
<style>
	p{
		font-size: 16px;
		line-height:1;
		margin-top:10px;
		margin-bottom: 15px;
	}
</style>
<p>AAA</p>
<p></p>
<p></p>
<p></p>
<p>BBB</p>

# margin负值的问题

+ margin-top和margin-left为负值，元素往上，往左移动
+ margin-right负值，右侧元素左移，自身不受影响
+ margin-bottom负值，下方元素上移，自身不受影响

# BFC理解和应用

+ Block format context，块级格式化上下文；
+ 一块独立渲染区域，内部元素的渲染不会影响边界以外的元素
+ 形成BFC的常见条件：
	+ float 不是 none
	+ position是 absolute 或 fixed
	+ overflow 不是 visible
	+ display是flex，inline-block

# float布局问题，以及clearfix（清除浮动）

+ 浮动布局会产生高度塌陷
+ 如何实现圣杯布局和双飞翼布局
	+ 特点：
		+ 三栏布局，中间一栏最先加载和渲染（内容最重要）
		+ 两侧内容固定，中间内容随着宽度自适应
		+ 一般用于PC网页
	+ 实现方法：
		+ 使用float布局
		+ 两侧使用margin负值，以便和中间内容重叠
		+  防止中间内容被两侧覆盖，一个用padding，一个用margin

<!--圣杯布局-->
<style>
	body{
	}
	#header{
	}
	#container {
		padding-left:200px;
		padding-right:150px;
	}
	#container .column{
		float:left;
	}
	#center{
		background-color: #ccc;
		width:100%
	}
	
	#left{
		
		background-color: yellow;
		width:200px;
		//让left和center重叠，left左移动
		margin-left:-100%;
		//相对自身移动到左面
		position:relative;
		right:200px
	}
	#right{
		background-color: red;
		width:150px;
		//自身不变，右面的元素右移，right块在外界看来没有宽度，所以浮动到上面
		margin-right:-150px;
	}
	#footer{
		clear:both;
		text-align:center;
		background-color: #f1f1f1;
	}
	/*手写clearfix：*/
	.clearfix:after{
		content: '';
		display: table;
		clear:both;
	}
</style>
<body>
	<div id = "header">this is header</div>
	<div id = "container" class="clearfix">
		<div id = "center" class = "column"> center</div>
		<div id = "left" class = "column"> left</div>
		<div id = "right" class = "column">right</div>
	</div>
	<div id = "footer">this is footer</div>
</body>

# flex画色子

- 实现一个三点的色子
- 常用语法回顾：
	- flex-direction:
	- justify-content:
	- align-items:
	- flex-wrap:
	- align-self:子元素

<!--flex画三个点的色子-->
<style>
	.box {
		width:200px;
		height:200px;
		border: 2px solid #ccc;
		border-radius: 10px;
		padding: 20px;
		display:flex;
		justify-content:space-between;/*两端对齐（水平）*/
	}
	.item{
		display:block;
		width:40px;
		height:40px;
		border-radius: 50%;
		background-color: #666;
	}
	.item:nth-child(2){
		align-self: center;/*第二项局中对齐，交叉轴(垂直)*/
	}
	.item:nth-child(3){
		align-self: flex-end;/*第三项：交叉轴的尾对齐*/
	}
</style>
<body>
	<div class="box">
		<div class = "item"></div>
		<div class = "item"></div>
		<div class = "item"></div>
	</div>
</body>

# 定位

* absolute和relative分别依据什么定位？
	+ relative依据自身定位，并且对外界元素不会有影响，通过top, left, 生效
	+ absolute依据最近一层的定位元素定位
* 居中对齐有哪些实现方式？
	+ 水平居中：
		+ inline元素：text-align：center
		+ block元素：margin：auto（左右两边自动填充）
		+ absolute元素：(必须知道子元素的宽度)left：50%（右移动父元素一半） + margin-left负值（向左拖拽调整）
	+ 垂直居中：
		+ inline元素：line-height 的值等于height的值
		+ absolute元素：(必须知道子元素的高度) top：50% + margin-top负值
		+ absolute元素： left:50%;right:50%;transform(-50%,-50%)---CSS3属性，注意兼容性
		+ absolute元素：（最终解决方案,不需要考虑浏览器兼容性和子元素的尺寸） top, left, bottom, right = 0 + margin:auto

# 图文样式

* line-height如何继承？
	+ 写具体数值，如30px，则继承该值
	+ 写比例，如2/1.5，则继承该比例
	+ 写百分比，如200%，继承计算出来的值，先计算后继承

<!--p标签的行高是多少？20px* 200% = 40px-->
<style>
body {
	font-size: 20px;
	line-height:200%;
}
p{
	font-size:16px;
}
</style>
<body>
	<p>AAA</p>
</body>

# CSS响应式

* rem是什么？
	+ rem是一个长度单位
		+ px是一个绝对的长度单位，最常用
		+ em是相对长度单位，相对于父元素，
		+ rem是相对长度单位，相对于根元素，常用于响应式布局 
* 如何实现响应式布局
	+ media-query,根据不同的屏幕宽度设置根元素font-size
	+ rem基于根元素定位
* rem的弊端：“阶梯”性
* 网页视口尺寸：
	* window.screen.height //屏幕高度
	* window.innerHeight //网页视口高度，除去头尾，与浏览器有关
	* document.body.clientHeight: //body高度
* vh网页视口高度的1/100window.innerHeight；
* vw网页视口宽度的1/100window.innerWidth；
* vmax取长宽两者最大值；vmin取长宽两者最小值

<style>
html {
/*根元素*/
	font-size:100px;
}
div{
	background-color:#ccc;
	margin-top:10px;
	/*0.16 * 100px*/
	font-size:0.16em;
}
</style>
<body>
	<p>rem1</p>
	<p>rem2</p>
	<p>rem3</p>
	<div>div1</div>
	<div>div2</div>
	<div>div3</div>
</body>
